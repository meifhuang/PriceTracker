# Price-Tracker

PriceTracker is a full-stack web application that automates price monitoring for pet products and manages personal cat expenses. The app scrapes product prices from major retailers (Petsmart, Petco, Chewy, Amazon), visualizes price trends over time, and provides a comprehensive expense tracking system with cashback calculations.

<a href="https://pricetracker-production-f4b9.up.railway.app/dashboard"> See live demo </a>

## Built With
* Flask
* Python
* PostgreSQL
* Jinja2
* Selenium + undetected_chromedriver
* Plotly.js
